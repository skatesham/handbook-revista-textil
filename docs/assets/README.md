# Assets

## Logo e Favicon

Para personalizar o handbook, adicione os seguintes arquivos nesta pasta:

- **logo.jpg** - Logo da Revista Têxtil (recomendado: 128x128px ou maior)
- **favicon.ico** - Ícone do navegador (recomendado: 32x32px)

## Formato recomendado

- Logo: JPG, PNG ou SVG
- Favicon: ICO ou PNG (16x16px ou 32x32px)

## Exemplo de geração

Você pode criar um favicon a partir do logo usando ferramentas online como:
- https://favicon.io/
- https://realfavicongenerator.net/
